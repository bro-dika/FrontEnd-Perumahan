# FrontEnd-Perumahan

Proyek multi‑halaman yang dirancang untuk menampilkan informasi developer perumahan secara profesional, modern, dan responsif. Tampilan UI/UX terinspirasi dari gaya website Positivus, dengan warna utama **hijau tua** dan latar **putih** agar terlihat elegan sekaligus bersih.

---

## Tema Desain

* **Primary Color:** Hijau tua (`#0B442F`)
* **Background:** Putih (`#FFFFFF`)
* **Accent:** Abu lembut
* **Font yang direkomendasikan:** Inter / Montserrat / Poppins

---

## Integrasi API (Opsional)

Jika ingin menambahkan backend:

* **GET /api/units** — mengambil daftar rumah
* **GET /api/units/:id** — detail unit
* **GET /api/artikel** — daftar artikel
* **POST /api/kontak** — kirim form kontak

---

## Pengembangan Lanjutan

Bebas untuk memperluas fitur:

* Halaman detail unit lengkap + slider
* Sistem booking unit
* Dark mode
* CMS untuk artikel (Strapi / Netlify CMS)
